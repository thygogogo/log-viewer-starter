# log-viewer-core

`log-viewer-core` 是 `log-viewer` 多模块中的共享模型模块，主要提供：

- `LogEntry`
- `CursorPageResponse`
- `PageResponse`
- `ServerConfig`

Starter 请使用：

- Spring Boot 2.x: `com.thy:log-viewer-boot2-starter`
- Spring Boot 3.x: `com.thy:log-viewer-boot3-starter`

完整说明见上级目录 `log-viewer/README.md`。
